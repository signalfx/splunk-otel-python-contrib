7. For the AI Assistant Team

To integrate with AI Defense, they need:
1. pip install cisco-aidefense-sdk
2. Choose API mode or Gateway mode
3. Add AIDefenseInstrumentor().instrument() for auto spans
4. Set env vars: AI_DEFENSE_API_KEY (API) or AI_DEFENSE_GATEWAY_URL (Gateway)
5. Reference: examples/multi_agent_travel_planner/ (API mode) or examples/gateway/multi_agent_travel_planner/ (Gateway mode)
What AI Defense SDK returns (InspectResponse):
• event_id — for correlation
• is_safe — boolean
• action — Allow / Block
• classifications — SECURITY_VIOLATION, PRIVACY_VIOLATION, SAFETY_VIOLATION
• rules — Prompt Injection, PII, Violence, etc.
• severity — NONE, LOW, MEDIUM, HIGH
Key gotcha: In API mode, the app decides what to do with is_safe=false. Even if the AI Defense policy is set to "monitor" (not "block"), the app can choose to block. The policy mode only matters in Gateway mode where the gateway enforces it.
